# Assets

Folder ini disiapkan untuk aset game (grafik, audio, font, dll).

Pada **Fase 1**, semua aset dibuat secara prosedural
(tekstur placeholder dari `Phaser.GameObjects.Graphics` di `PreloadScene`),
sehingga folder ini sengaja kosong.

Pada **Fase 3** (deployment ke Play Store), folder ini akan berisi
sprite, tileset, audio, dan font final.
